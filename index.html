Embedded Systems Eye Opener

A smart IoT system that detects open windows and indoor conditions, preventing unnecessary heating energy waste.

The Problem

Heating energy is often wasted when windows are left open while heating is on. This system automatically detects this situation and reacts intelligently.

The Solution

By fusing temperature, humidity, and window state data, the system provides real-time recommendations and heater control.

System Overview

The system consists of an embedded sensor node, an MQTT communication layer, and a web-based user interface for monitoring and control.

Hardware & Sensors

A Raspberry Pi collects temperature, humidity, and window state data using I2C sensors and publishes them via MQTT.

Web Interface

A real-time dashboard visualises sensor data and allows heater control through MQTT over WebSocket.

Future Work

Future extensions include multi-room support, adaptive decision logic, and integration with smart heating systems.
