The Problem

Heating energy is frequently wasted when windows are left open while heating systems remain active. This often happens unintentionally, leading to increased costs and reduced energy efficiency.

The Solution

Embedded Systems Eye Opener continuously monitors indoor temperature, humidity, and window state. By fusing sensor data, the system detects inefficient conditions and responds with clear recommendations or automatic heater control.

This approach reduces heating costs, improves energy efficiency, and increases user awareness with minimal user intervention.

System Overview

The system consists of an embedded Raspberry Pi sensor node, an MQTT-based communication layer, and a web-based user interface. Sensor data is sampled, transmitted, processed, and visualised in real time.

Future Work

Future development may include multi-room deployment, adaptive decision thresholds using historical data, integration with smart heating systems, and enhanced robustness during network outages.
